# InAmigos Foundation Awareness Webpage

## Project Overview

This project is a responsive NGO awareness webpage developed for the InAmigos Foundation internship task.

The purpose of this webpage is to create awareness about the foundation's mission, ongoing projects, social impact, and opportunities for community participation.

---

## Features

* Introduction to InAmigos Foundation
* Information about ongoing projects
* Social impact statistics
* Purpose and mission section
* Call-to-action section
* Contact information
* Clean and responsive design using HTML and CSS

---

## Technologies Used

* HTML5
* CSS3

---

## Ongoing Projects Included

* Project Seva
* Project Bachpanshala
* Project Jeev
* Project Udaan
* Project Prakriti
* Project Vikas

---

## Social Impact Highlights

* 50,000+ Meals Distributed
* 2,000+ Children Educated
* 900+ Girls Empowered
* 20,000+ Trees Planted
* 50+ Animals Fed Daily
* 30,000+ Interns Trained

---

## Purpose

The webpage aims to spread awareness about the foundation's efforts in education, food security, women empowerment, environmental conservation, animal welfare, and skill development.

## Submission

This project was developed as part of the NGO Awareness Webpage Creation Task assigned during the InAmigos Foundation Web Developer Internship.
